# Analyzing Political Discourse on Twitter

This project analyzes how members of the U.S. Congress used Twitter between **October 2012 and December 2014**, focusing on **issue salience, engagement patterns, partisan framing, and information flow**.  
The analysis combines **text analytics, temporal trends, and social network analysis** to understand how political narratives emerged and spread on social media.
